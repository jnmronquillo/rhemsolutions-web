Software a Medida

Scrum

Proceso de Conceptualizar

Se crea una visión del producto y se determina quienes llevaran acabo el proyecto

Proceso de Especulación

Se determinan los costes y la agenda, se cierra la primera aproximación de lo que se puede producir

Proceso de Exploración

Se desarrolla un incremento del producto (Sprint n)

Proceso de Revisión

Equipo y usuarios revisan lo construido

Proceso de Cierre

Se entrega una versión del producto

Programacion Extema

Proceso de diseño

Se diseña la interfaz de usuario

Proceso de implementación

Se escribe el código necesario para implementar la funcionalidad

Proceso de pruebas

El personal de pruebas se encarga de encontrar errores en el sistema

Proceso de depuración

El personal de desarrollo se encarga de corregir los errores del sistema

Proceso de Implementación

El personal de desarrollo sube al servidor web la aplicación con la nueva funcionalidad
